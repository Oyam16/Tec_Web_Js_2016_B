## Enfasis de Texto

* Para nosotros utilizar un texto en **negrita** se utilizan dos "**" (asteriscos).
* Para nosotros utilizar un texto en *cursiva* vamos a utilizar 1 "*" (asterisco).
* Para nosotros ~~tachar un texto~~ necesitas usar "~" (tildes).
